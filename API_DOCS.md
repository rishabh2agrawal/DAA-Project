# ClimateIQ API Documentation

Base URL: `http://127.0.0.1:8000`

The backend is implemented with FastAPI in `backend/app.py`. Prediction endpoints use classical ML models when model files are available and controlled heuristic fallbacks otherwise.

## Request Schema

Most prediction routes accept:

```json
{
  "temperature": 28.0,
  "humidity": 67.0,
  "rainfall": 2.4,
  "aqi": 115.0,
  "wind_speed": 8.0,
  "historical_rainfall": 4.0,
  "NDVI": 0.3,
  "NDBI": 0.35,
  "NDWI": 0.25,
  "heat_index": 34.7,
  "rainfall_anomaly": 0.5,
  "uhi_score": 3.0
}
```

Only `humidity`, `rainfall`, and `aqi` are strictly required by the schema; defaults are supplied for the other project-ready fields.

## System Endpoints

### `GET /weather`

Returns the latest valid processed weather row.

### `GET /model_metrics`

Returns the latest model metrics generated by:

```bash
python ai_models/train_extended.py
```

### `GET /climate_summary`

Returns date range, average climate values, and a short method note for the processed dataset.

## Prediction Endpoints

### `POST /predict_temperature`

Returns:

```json
{
  "predicted_temperature": 29.1,
  "temperature_class": "Hot",
  "horizon": "next-day estimate"
}
```

Backward-compatible alias: `POST /predict`

### `POST /predict_aqi`

Returns:

```json
{
  "predicted_aqi": 102.0,
  "aqi_class": "Poor",
  "horizon": "next-day estimate"
}
```

### `POST /predict_rainfall`

Returns one of: `No Rain`, `Light`, `Moderate`, `Heavy`, `Extreme`.

```json
{
  "category": "Light",
  "horizon": "next-day class estimate"
}
```

### `POST /predict_heatwave`

Returns:

```json
{ "risk": "Moderate" }
```

### `POST /predict_flood`

Returns:

```json
{ "risk": "Low" }
```

### `POST /predict_dengue`

Returns:

```json
{ "risk": "Low" }
```

### `POST /predict_human_risk`

Returns:

```json
{
  "risk": "Moderate",
  "method_note": "Risk is an ML/rule estimate for awareness, not medical advice."
}
```

Backward-compatible aliases:

- `POST /health_risk`
- `POST /predict_risk`

## Method Notes

- Weather comes from the accessible project dataset.
- NDVI, NDBI, NDWI, LST, UHI, and AQI are proxy/generated in v1 unless official datasets are added.
- Predictions are next-day or class estimates, not official government warnings.
